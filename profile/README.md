## Overview

Swish Advanced is an elegant window management application that uses trackpad gestures to snap, resize, minimize, and organize windows with fluid, intuitive movements that feel like a natural extension of the macOS trackpad experience. It replaces manual window edge dragging with swipe gestures on the titlebar that snap windows to halves, quarters, thirds, and other positions instantly.

Titlebar gestures transform the titlebar into a gesture surface — swipe up to maximize, down to minimize, left to snap left half, right to snap right half, diagonal for quarters. Two-finger pinch centers the window, and spread expands it. The gestures are discoverable because they match the spatial direction of the movement they produce.

Window snapping places windows into precise screen positions with smooth animations. The snap positions cover the most common arrangements, and the grid system handles irregular layouts beyond simple halves and quarters.

Multi-display awareness handles management across monitors with gestures that move windows between displays matching the physical layout. Cross-display management is as fluid as single-display.

For trackpad users who want window management as natural as other macOS gestures, Swish Advanced turns titlebar swipes into instant positioning with the elegance macOS users expect.

## Performance

Swish balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
